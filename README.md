# Cotação do Dólar - Exercício de Conversão de Moeda

Este programa em C# é um exercício de conversão de moeda, que permite ao usuário calcular o valor em reais correspondente a uma determinada quantia em dólares com base na cotação fornecida.

## Como usar o programa

1. **Iniciar o Programa:**
   Execute o programa em um ambiente de desenvolvimento C# de sua escolha.

2. **Fornecer a Cotação Atual:**
   Ao iniciar o programa, será solicitado que você forneça a cotação atual do dólar em relação ao real.

3. **Informar a Quantidade de Dólares:**
   Após fornecer a cotação, você será solicitado a informar a quantidade de dólares que deseja converter para reais.

4. **Obter o Valor em Reais:**
   O programa calculará o valor correspondente em reais com base na quantidade de dólares fornecida e na cotação informada, incluindo um IOF de 6%.

5. **Decidir se Deseja Continuar:**
   Após a conversão, o programa perguntará se deseja realizar outra operação de conversão. Se sim, você pode fornecer outra quantia em dólares e repetir o processo. Se não, o programa será encerrado.

## Resumo do Programa

O programa é um simples conversor de moeda que recebe a cotação do dólar, a quantidade de dólares a serem convertidos e calcula o valor correspondente em reais, incluindo um IOF de 6%. O usuário pode repetir a operação quantas vezes desejar, até optar por encerrar o programa.
